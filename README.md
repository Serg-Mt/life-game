# Conway's Game of Life

Реализация клеточного автомата без фреймворков и зависимостей кроме сборочных.  
[**→ Live demo**](https://life-game-js.vercel.app)

## Архитектура

Паттерн **Strategy** с двумя независимыми осями замены: алгоритм симуляции и способ отрисовки. Любую комбинацию можно переключить через UI в рантайме — без остановки симуляции.
Добавление нового алгоритма или рендерера сводится к реализации одного интерфейса.

## Стек

`TypeScript` · `Vite` (Rolldown) · `Sass` · `Pug` · `WebGL 2.0`  
Zero runtime dependencies.
